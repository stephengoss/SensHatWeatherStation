# Pi Weather Station
Its a python script for a raspberry pi sens hat weather stations
Switches to an external temperature sensor, when it detects one.
Uploads data to an ibm cloud database,
Temperature is smoothed to give more accurate readings
Indication of temperature trends with coloured symbols
Based on code originaly created by John M. Wargo [https://johnwargo.github.io/](https://github.com/johnwargo/pi-weather-station)
